Start menu:
    use arrow keys to move selection across cards (yellow indicator)
    press space to select/unselect card (selected cards have green indicator)
    press enter to start game
    4 cards selected max
    press tab to cycle through bot difficulty levels (off, easy, medium, hard)
In game:
    use number keys to spawn enemies (the order is shown with the card icons)
    1, 2, 3, 4 for the left side
    7, 8, 9, 0 for the right side
Game over:
    press space to play again
